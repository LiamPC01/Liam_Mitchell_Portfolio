Library App

Project Objective

Create a user-friendly web application for cataloging books, inputting detailed information (title, author, number of pages), and tracking reading status. Users can view their collection in a personalized virtual library.

The Process

  • Analyzed similar web applications for inspiration.
  • Created initial sketches in Figma.
  • Built HTML5 structure for semantic and accessible markup.
  • Implemented JavaScript functionalities (adding books, updating information, tracking status).
  • Applied CSS3 for styling, focusing on visual appeal and responsive design.

The Outcome

  • Designed and developed a fully functional, visually appealing web application.
  • Features an intuitive interface with smooth interactions and personalized experience.
